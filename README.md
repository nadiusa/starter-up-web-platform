# starter-up-web-platform
### This is a secure REST-ful API with Spring Boot, Security, and Data MongoDB.
##### It is a project for developing the licence thesis, which is a web platform for crowdfunding, where an entrepreneur is able to add his existing business that needs an investment, as well as a simple creator that wants to develop a start-up and needs material help to implement a great idea into reality.

##### The project contains just the backend part of the project, therefore here you can find:


-microservices for user authentication:

- user can sign up;
- user can sign in;
- user can log out;

-user can update his/her password;

-user can reset password, by sending an email with a reset token link;

-microservices for:

- project creation;
- project updating;
- project deleting;
- project reading/getting;
- getting the list of all projects;
